# Events

The AG-UI Dart SDK models protocol traffic as a sealed set of strongly typed events. Each event extends `BaseEvent`.

## BaseEvent

```dart
sealed class BaseEvent extends AGUIModel with TypeDiscriminator {
  final EventType eventType;
  final int? timestamp;
  final Object? rawEvent;

  const BaseEvent({
    required this.eventType,
    this.timestamp,
    this.rawEvent,
  });

  String get type => eventType.value;
}
```

### Example

```dart
await for (final event in client.runAgent('agent', input)) {
  switch (event) {
    case RunStartedEvent(:final runId):
      print('Started run: $runId');

    case StepStartedEvent(:final stepName):
      print('Starting step: $stepName');

    case RunFinishedEvent(:final result, :final outcome):
      print('Finished run with result=$result outcome=${outcome?.type}');

    case RunErrorEvent(:final message):
      print('Run failed: $message');
  }
}
```

### TextMessageChunkEvent

Chunk events are still part of the protocol, but they are usually normalized into start/content/end events before application.

```dart
class TextMessageChunkEvent extends BaseEvent {
  final String? messageId;
  final TextMessageRole? role;
  final String? delta;
  final String? name;

  const TextMessageChunkEvent({
    this.messageId,
    this.role,
    this.delta,
    this.name,
    int? timestamp,
  });
}
```

### Example

```dart
final buffer = StringBuffer();
String? currentMessageId;

await for (final event in stream) {
  switch (event) {
    case TextMessageStartEvent(:final messageId):
      currentMessageId = messageId;
      buffer.clear();

    case TextMessageContentEvent(:final messageId, :final delta):
      if (messageId == currentMessageId) {
        buffer.write(delta);
      }

    case TextMessageEndEvent(:final messageId):
      if (messageId == currentMessageId) {
        print(buffer.toString());
      }
  }
}
```

## Event Handling Patterns

### Pattern Matching

```dart
switch (event) {
  case TextMessageStartEvent():
    onMessageStarted(event);
  case TextMessageContentEvent():
    onMessageDelta(event);
  case TextMessageEndEvent():
    onMessageFinished(event);
  case ToolCallStartEvent():
    onToolCallStarted(event);
  case ToolCallArgsEvent():
    onToolCallArgs(event);
  case ToolCallEndEvent():
    onToolCallEnd(event);
  case ToolCallResultEvent():
    onToolCallResult(event);
  case StateSnapshotEvent():
    onStateSnapshot(event);
  case StateDeltaEvent():
    onStateDelta(event);
  case MessagesSnapshotEvent():
    onMessagesSnapshot(event);
  case RunErrorEvent():
    onError(event);
  case RawEvent():
    onRawEvent(event);
  case CustomEvent():
    onCustom(event);
  default:
    print('Unhandled event: ${event.type}');
}
```

### Stream Helpers

```dart
extension EventStreamExtensions on Stream<BaseEvent> {
  Stream<String> get textContent =>
      whereType<TextMessageContentEvent>().map((e) => e.delta);

  Stream<ToolCall> get toolResults =>
      whereType<ToolCallResultEvent>().map((e) => e.toolCall);

  Stream<RunErrorEvent> get errors =>
      whereType<RunErrorEvent>();
}
```

## Testing

```dart
class TestEvents {
  static RunStartedEvent runStarted([String? runId]) =>
      RunStartedEvent(
        threadId: 'thread-1',
        runId: runId ?? 'test-run',
        timestamp: DateTime.now().millisecondsSinceEpoch,
      );

  static TextMessageContentEvent textDelta(String text, [String? messageId]) =>
      TextMessageContentEvent(
        messageId: messageId ?? 'test-msg',
        delta: text,
        timestamp: DateTime.now().millisecondsSinceEpoch,
      );
}
```
